# To-Do List App

A simple web application that allows users to manage daily tasks. Built with Node.js, Express, and MySQL.

## Features

- Add new tasks
- Mark tasks as complete or incomplete
- Delete tasks
- Filter tasks by status

## Technologies Used

- Node.js
- Express.js
- MySQL
- HTML / CSS / JavaScript

## Installation

### Prerequisites

- MySQL
